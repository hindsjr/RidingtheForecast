# Riding the Forcast

By Julia Hinds

## Overview
This project analyzes how weather conditions influence bike-share ridership in Chicago using August 2025 trip data from Divvy Bikes combined with hourly weather observations. The goal is to translate data insights into actionable operational and planning decisions.

NOTE: Not finalized!

### Key Questions
How do temperature and precipitation affect bike-share demand?

Do member and casual riders respond differently to weather?

Can weather forecasts help anticipate demand spikes?

## Tools and Libraries
Python (Pandas, Matplotlib, Seaborn)

Datasets: 
- Trip Data — bike_data.csv — Ride data for August 2025 - pulled from Divvy Bikes public data: https://divvybikes.com/system-data
- Hourly Chicago weather data for August 2025 — weather_data.csv - pulled from Open-Meteo api
